## 🇧🇷 Português Brasileiro

### Sobre o Projeto

**Raízes & Café** é um projeto desenvolvido como parte da terceira fase do programa SENAI de Tecnologia para Desenvolvimento de Jogos (TPGD).

Este é um projeto interativo desenvolvido em Unity que combina elementos de narrativa, diálogos e mecânicas de gameplay para criar uma experiência envolvente.

### 📋 Requisitos

- **Unity** 2022 ou superior
- **Visual Studio** ou outro editor de C# compatível
- **Git** para controle de versão

### 🚀 Como Começar

1. Clone o repositório:
```bash
git clone <URL_DO_REPOSITORIO>
```

2. Abra o projeto no Unity:
   - Abra o Unity Hub
   - Clique em "Add Project"
   - Selecione a pasta do projeto

3. Aguarde a compilação e importação de assets

### 📁 Estrutura do Projeto

```
Assets/
├── Audio/              # Arquivos de áudio e música
├── Dialogues/          # Sistemas de diálogo
├── Fields/             # Cenários e ambientes
├── Images/             # Sprites e texturas
├── Prefabs/            # Prefabs reutilizáveis
├── Scenes/             # Cenas do jogo
├── Scripts/            # Código C#
├── Settings/           # Configurações do projeto
└── TextMesh Pro/       # Recursos de fontes
```

### 🛠️ Tecnologias Utilizadas

- **Unity Engine** - Engine de desenvolvimento
- **C#** - Linguagem de programação
- **Fungus** - Framework para narrativa visual
- **Universal Render Pipeline** - Sistema de renderização
- **Input System** - Sistema de entrada moderno

### 📝 Licença

Este projeto foi desenvolvido como material educacional do programa SENAI.

## 🇬🇧 English

### About the Project

**Raízes & Café** is a project developed as part of the third phase of SENAI's Game Development Technology Program (TPGD).

This is an interactive project developed in Unity that combines narrative elements, dialogue systems, and gameplay mechanics to create an engaging experience.

### 📋 Requirements

- **Unity** 2022 or newer
- **Visual Studio** or compatible C# editor
- **Git** for version control

### 🚀 Getting Started

1. Clone the repository:
```bash
git clone <REPOSITORY_URL>
```

2. Open the project in Unity:
   - Open Unity Hub
   - Click "Add Project"
   - Select the project folder

3. Wait for compilation and asset import

### 📁 Project Structure

```
Assets/
├── Audio/              # Sound and music files
├── Dialogues/          # Dialogue systems
├── Fields/             # Scenarios and environments
├── Images/             # Sprites and textures
├── Prefabs/            # Reusable prefabs
├── Scenes/             # Game scenes
├── Scripts/            # C# code
├── Settings/           # Project settings
└── TextMesh Pro/       # Font resources
```
